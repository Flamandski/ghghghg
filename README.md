# Random Task Generator

**Автор:** Груздев Роман Сергеевич

**Дата:** [2026-04-28]

## Описание

Random Task Generator — это десктоп‑приложение с графическим интерфейсом на Python, которое помогает бороться с прокрастинацией и разнообразить повседневные задачи. Программа случайным образом выбирает задачу из списка, сохраняет историю всех сгенерированных заданий и позволяет фильтровать их по категориям.

## Основные возможности

- Генерация случайных задач по нажатию кнопки
- Категории задач: учёба, спорт, работа
- Сохранение истории с временными метками
- Фильтрация истории по категориям
- Автоматическое сохранение в JSON‑файл
- Очистка истории с подтверждением
